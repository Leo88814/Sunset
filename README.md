# 前台功能

1.顯示已上映與即將上映電影列表

2.主頁、品牌介紹、Q&A

3.註冊與登入，登入後可訂票與使用會員專區

4.會員專區能更改密碼、更改個人資料、確認跟取消已訂購電影、查看歷史訂單並評分、可進行儲值與查看儲值紀錄

5.訂票流程選擇電影、選擇日期、選擇場次、確認訂單

# 後台功能

1.電影的新增與編輯

2.場次的新增與刪除

3.劇照新增與刪除

4.會員帳號管理

5.權限控管
